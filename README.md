## Installation

1. Install node.js and npm
2. Clone the repo
   ```sh
   git clone https://github.com/cristhianbenitez/face-recognition-app.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run the app localy

```sh
 npm start
```
